# 🌾 Agri Rental System

A full-stack web application that enables farmers to rent and manage agricultural tools online in a simple, efficient, and digital way. The platform connects tool owners and farmers, helping improve accessibility and reduce manual dependency in agriculture.

---

## 🚀 Features

- 👤 User authentication (Register / Login)
- 🌱 Browse agricultural tools
- 🛠️ Add, update, and manage rental tools
- 📅 Book tools for specific time periods
- 📦 Booking management system
- 🖼️ Image upload for tools
- 🔐 Secure REST API backend
- 👨‍🌾 Separate user and admin flow (if implemented)

## 🌐 API Endpoints

| Method | Endpoint            | Description             |
|--------|---------------------|-------------------------|
| POST   | /api/auth/register  | Register new user       |
| POST   | /api/auth/login     | Login user              |
| GET    | /api/tools          | Get all tools           |
| POST   | /api/tools          | Add new tool            |
| POST   | /api/bookings       | Create booking          |
